## Key Features

* Booster Pack Simulator for all 9 base booster sets.
* Card Collection **WIP**
* Real-time Price Fetching **WIP**
* Dark/Light mode **WIP**
  
## How To Use

To install the app, allow downloads from third-party sources in your Android device.
